PRACTICE JAVA SCRIPT

When you're checking words like "cat" or "dog", they are strings. Without quotes, JavaScript thinks cat is a variable name.

Term What It Is

Function
A block of code that does something
Parameter
The placeholder when defining a function (e.g. function sayHi(name) → name is a parameter)
Argument
The actual value you pass to a function (e.g. sayHi("Ali") → "Ali" is the argument)
