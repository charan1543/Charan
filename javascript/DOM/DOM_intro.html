It's Dom intro

The DOM (Document Object Model) in JavaScript is a programming interface that allows you to interact with and manipulate HTML and XML documents.
When a web page loads, the browser creates a DOM of the page, which you can use with JavaScript to dynamically:

  • Change HTML elements and their contents
  • Modify CSS styles
  • Add or remove elements
  • Handle events like clicks, mouse movements, form submissions, etc.
